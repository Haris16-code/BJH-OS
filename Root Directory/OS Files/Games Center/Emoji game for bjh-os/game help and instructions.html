                                            HELP AND INSTRUCTIONS

Emoji Finding Game is a game in which we find different emojis.

This game is created by BJH OS TEAM.

HOW TO PLAY THIS GAME?

1. FOR EXAMPLE, YOU CAN PLAY THE FIRST LEVEL WHERE THERE ARE MANY EMOJIS. YOU CAN FIND DIFFERENT EMOJI BY CLICKING ON THEM, AND THE NEXT LEVEL WILL AUTOMATICALLY OPEN.

HOW TO OPEN THE LEVEL LIST? OR WHAT IS THE KEY OF THE LEVEL LIST?

WHEN YOU REACH LEVEL NO 5, WE PROVIDE A LEVEL LIST KEY. USE THE KEY IN THE LEVEL LIST SECTION.

HOW TO UPDATE THE GAME?

JUST PRESS ON THE GAME UPDATE ICON  WHEN YOU CLICK ON THIS ICON, A GAME UPDATE SECTION WILL OPEN. IN THIS SECTION, YOU CAN SEE THE UPDATE INFO.

WHICH CAN BE ADDED IN THE GAME AFTER UPDATE?

THERE ARE MANY THINGS THAT ARE ADDED TO THE GAME AFTER AN UPDATE, SUCH AS LEVELS, GAME UI, AND MANY MORE THINGS.

HOW TO INSTALL THE UPDATE?

JUST PASTE THE DOWNLOADED FILES FROM THE GAME UPDATE SECTION INTO THE EMOJI GAME FOLDER, WHICH IS LOCATED IN THE BJH OS FOLDER. THEN THE GAME WILL BE UPDATED.
